# 番茄钟手机部署指南

## 方法一：局域网快速访问（手机电脑需连同一 WiFi）

这是最快的方法，不需要注册任何账号，只要电脑开着就能用。

1. **确保电脑服务已启动**
   - 在项目目录 `d:\06.AI项目\pomodoro` 下。
   - 终端运行：`python -m http.server 8000`

2. **获取访问地址**
   - 你的电脑当前 IP 为：`175.185.98.217`
   - 手机浏览器输入：`http://175.185.98.217:8000`

3. **安装到手机（像 App 一样使用）**
   - **iOS (Safari)**：点击底部中间的“分享”按钮 → 下滑找到并点击“添加到主屏幕”。
   - **Android (Chrome)**：点击右上角三个点菜单 → 点击“安装应用”或“添加到主屏幕”。
   - *这样你的桌面上就会出现一个“番茄钟”图标，点击即可全屏运行，体验和原生 App 一样！*

## 方法二：使用 Vercel 免费部署（推荐）

这种方法可以让你的番茄钟生成一个永久网址（如 `https://my-pomodoro.vercel.app`），**不需要电脑开机**，任何网络（4G/5G）都能访问。

1. **准备工作**
   - 注册一个 [GitHub](https://github.com/) 账号。
   - 注册一个 [Vercel](https://vercel.com/) 账号（可以用 GitHub 登录）。

2. **部署步骤**
   - 将本项目上传到你的 GitHub 仓库。
   - 登录 Vercel，点击 "Add New..." -> "Project"。
   - 选择你刚才上传的 GitHub 仓库，点击 "Import"。
   - 点击 "Deploy"。
   - 等待几秒钟，Vercel 会给你一个 `.vercel.app` 的网址。

3. **分享与安装**
   - 把这个网址发到手机上打开。
   - 同样执行“添加到主屏幕”操作。

## 常见问题

- **手机打不开网页？**
  - 检查手机和电脑是否连的同一个 WiFi。
  - 检查电脑防火墙是否允许了 Python 网络通信。
  - 尝试关闭电脑防火墙测试一下。

- **没有提示音？**
  - 手机浏览器为了省电，有时会限制自动播放声音。
  - 第一次打开时，请随便点击一下页面（比如“开始”按钮），让浏览器获得播放权限。
  - 确保手机没有处于静音模式。
